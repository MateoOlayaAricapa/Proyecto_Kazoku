# Proyecto_Kazoku
## Descripción: 
Este proyecto abarca el desarrollo de un sitio web para un restaurante japonés ubicado en la ciudad de Cali, Colombia. 

## Tecnologías:
Se utilizó React js, .NET CORE (entity framework), PostgreSQL y Figma.
